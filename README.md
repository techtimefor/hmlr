## Hannah Montana Linux Revived (hmlr)

A revived version of the ancient Hannah Montana Kubuntu based distro

## Downloads

Check releases for latest iso avaliable or you can manually build it following the building process.

## Building Process

> DO NOT RUN THE SCRIPT AS ROOT OR SUDO YOURSELF

1. git clone https://github.com/techtimefor/hmlr.git
2. cd hmlr.git
3. chmod +x hmlr-builder.sh
4. ./hmlr-builder.sh
